luna-libspotify-dotnet

A C++/CLI Library which provides managed class bindings for using libspotify in dotnet projects

Download this project as a .zip file Download this project as a tar.gz file

If you want to create an own spotify player, a headless spotify service or what ever you can imagine in dotnet, luna-libspotify-dotnet will provide you easy, managed access to the native spotify api!
Just add a reference and make sure to place "libspotify.dll" right beside your executable and start playing around with it - you don't have to worry about memory management, pointers or any native stuff, just use your C#, XAML or WinForms experience and start programming your application!

See the implementation status here: https://github.com/Lunatix89/luna-libspotify-dotnet/wiki

If you want to know, how it can be done, take a look at the DemoPlayer project (not a beatifull player, it is just for testing and a little bit ugly, but it is good enough for testing purposes).

Have fun!
